# Online-Shoppingmall-Minigame
--------------------------------
 
 ## 프로젝트 설명
--------------------------------
 JavaScript를 이용한 동적 요소 생성, 버튼 동작 등의 기능 구현 및 저장된 데이터를 브라우저에 표시하는 것 등을 연습하기 위한 프로젝트
 데이터 저장형식은 초기에는 배열을 사용하다가 JSON으로 변경
